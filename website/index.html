Фракталы и их применения

Мнимые чилса

Мнимые числа были предложенны в XVII веке французским математиком Рене Декарт. Мнимые числа выглядят как одночлен:xi; Пример мнимого числа: 4i. Однако i - не просто переменная, а константа, i*i = -1 Данные числа можно изобразить по оси ординат в числовой плоскости, действительные числа - по оси абсцисс.

Комплексные чилса

Комплексные числа выглядят как многочлен: a + bi, где a - действительная часть, i - мнимая единица. Комплексные числа можно представить как точку на числовой плоскости. Пример комплексного числа: 12 + 71i

Множество Жюлиа

Если взять комплексную (числовую) плоскось и для каждой точки определить, стремиться ли она к нулю после n итреаций f(z)=z*z И закрашивать её в зависимости от этого, можно получить круг. Все числа больше 1 будут стремиться к бесконечности, меньше - к нулю. Если добавить число c к формуле, то мы получим z=z*z+c. Ипользуя такую формулу можно увидить, что множество начало видоизменяться.

Множество Мандельброта

Множество мандельброта состволяется по такой-же формуле, однако c = yi + x и z в нулевое = 0.
В данное можнство можно приближаться бескоечное количество времени

4-х мерные фракталы

Четырёхмерные фракталы основаны на кватернионах
Кватернио́ны (от лат. quaterni, по четыре) — система гиперкомплексных чисел, образующая векторное пространство размерностью четыре

Применения фракталов

Фракталы широко применяются в компьютерной графике для
построения изображений природных объектов, таких, как деревья,
кусты, горные ландшафты, поверхности морей и т.д. Существует множество программ,
служащих для генерации фрактальных изображений.
